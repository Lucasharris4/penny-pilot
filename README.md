# Penny Pilot

A self-hosted, privacy-first personal finance tracker.

## Connecting your bank

Penny Pilot syncs transactions via [SimpleFIN](https://www.simplefin.org/), a privacy-focused aggregator that never exposes your bank credentials to Penny Pilot. See [`docs/simplefin-setup.md`](docs/simplefin-setup.md) for a step-by-step walkthrough of the one-time setup.
